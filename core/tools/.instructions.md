# 🔧 core/tools/ - AI 工具集規範

定義 AI Agent 可以調用的技能（LangChain Tools）之標準實作方式。

---

## 🏗️ 工具開發準則

1. **唯一出口**：所有工具必須透過 `core/tools/__init__.py` 的 `ALL_TOOLS` 統一匯出。
2. **字串回傳**：回傳值類型一律為 `str`。工具內部應捕獲所有異常並回傳格式化錯誤字串。
3. **自我描述 (Critical)**：Docstring 是 Agent 決策的依據，必須包含：
   - **使用場景**：Agent 什麼時候應該選用此工具。
   - **參數說明**：詳細描述輸入參數的限制。
   - **Agent 提示**：針對 LLM 的特殊調用指導。

---

## 🔧 新增工具流程

1. **按用途命名**：根據工具性質命名檔案（如 `search_*.py`, `pdf_*.py`）。
2. **封裝邏輯**：使用 `@tool` 裝飾器，並將複雜邏輯提取為私有函數（`_` 開頭）。
3. **即插即用**：在 `__init__.py` 中註冊後即可在 Workflow 中使用。
